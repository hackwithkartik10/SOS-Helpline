# SOS Helpline

🚨 **SOS Helpline** is a simple Python-based tool that provides quick access to important emergency and public service helpline numbers.

## Features

- Access emergency contact numbers instantly
- Easy-to-use command-line interface
- Lightweight and fast
- Useful for students, travelers, and everyday users
- Organized list of essential helpline services

## Available Services

Examples may include:

- Police Emergency
- Ambulance
- Fire Brigade
- Women Helpline
- Child Helpline
- Disaster Management
- Medical Emergency
- Cyber Crime Helpline

## Installation

```bash
git clone https://github.com/hackwithkartik10/SOS-Helpline.git
cd SOS-Helpline
```

## Requirements

- Python 3.8 or higher

## Usage

```bash
python3 emergancy.py
```

Example Output:

```text
===== SOS HELPLINE =====

5. Cyber Crime Helpline

Enter Service Name : Police

Output: 100
```
